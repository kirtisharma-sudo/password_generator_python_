# 🔐 Password Generator

A beginner-friendly Python application that generates secure, random passwords using customizable lengths and a mix of letters, numbers, and special characters.

![Python](https://img.shields.io/badge/Python-3.12-blue.svg)
![CLI](https://img.shields.io/badge/Application-CLI-success)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

---

## 📖 Description

This project was built to practice Python fundamentals including:

- Random number generation
- String manipulation
- User input handling
- Functions
- Modular programming

The program generates strong passwords based on the user’s desired length using uppercase letters, lowercase letters, digits, and special symbols.

---

## ✨ Features

- 🔢 User-defined password length
- 🔐 Generates secure random passwords
- 🔡 Includes uppercase and lowercase letters
- 🔢 Includes numbers
- ✨ Includes special characters
- 🖥️ Simple command-line interface

---

## 🛠️ Technologies Used

- Python 3
- `random` module
- `string` module

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/kirtisharma-sudo/password_generator_python_.git
```

### Navigate to the project

```bash
cd password_generator_python_
```

### Run the program

```bash
python password_generator.py
```

*(Replace `password_generator.py` with your actual Python filename if it's different.)*
